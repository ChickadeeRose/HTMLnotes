There are 3 ways to do color
- Name
- Hex
- Styles using RGBA

~ Name is limited to 140 defined names that you can lookup

~ If you use hex you have 16,777,216 colors
- Web-safe colors use repeated values
- With syntax you are going to use a # in front of the color
- The values used are 1 2 3 4 5 6 7 8 9 0 a b c d e f

~ RGBA stands for Red, Green, Blue, Alpha
- These can only be used with stylesheets
- Syntax is: rgba(0-255, 0-255, 0-255, 0-1)
